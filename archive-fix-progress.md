## 📋 存档规范

### 命名格式

**日报：** `[代码]-YYYY-MM-DD.md`
- 示例：`B1-2026-03-07.md`

**周报：** `[代码]-周报-YYYY-Www.md`
- 示例：`D2-周报-2026-W10.md`

**月报：** `[代码]-月报-YYYY-MM.md`
- 示例：`E3-月报-2026-03.md`

### 存档位置

```
/root/.openclaw/workspace/archive/[代码]-[任务名称]/
```

### 存档步骤（每个监控任务必须执行）

1. **生成报告**
2. **保存到本地**（使用 write 工具）
3. **发送飞书**（使用 message 工具）
