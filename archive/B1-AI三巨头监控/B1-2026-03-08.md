### 2. 🛒 Anthropic 推出 Claude Marketplace（2026-03-07）

**事件**：Anthropic 正式推出 Claude Marketplace，允许企业客户使用现有的 API 承诺额度购买基于 Claude 的第三方软件应用。

**关键细节**：
- 企业客户可以将已承诺的年度 Anthropic API 和服务支出的**一部分**用于购买第三方应用
- **Anthropic 不收取佣金**，直接由合作伙伴向客户收费
- 初期有**6 个合作伙伴**，包括 AI 数据标注、文档处理、客户支持工具等
- 旨在通过**单一关系模型**整合企业云支出

**影响**：🟢 **中等**
- 商业模式创新：从纯 API 提供商转向平台生态
- 对抗 OpenAI 的 GPT Store 策略
- 降低企业采购门槛，加速 Claude 在企业市场渗透

**来源**：[MLQ.ai](https://mlq.ai/news/anthropic-launches-claude-marketplace-to-channel-enterprise-ai-budgets-into-partner-apps/) | [The Next Web](https://thenextweb.com/news/anthropic-marketplace-claude-enterprise-software)

### 3. 🎖️ Anthropic 被指定为"国家安全供应链风险"（2026-03-07）

**事件**：美国国防部长 Pete Hegseth 将 Anthropic 指定为"国家安全供应链风险"，此前该公司拒绝在完全自主武器系统中使用其 AI 技术。

**关键细节**：
- 冲突源于"金顶"导弹防御计划（Golden Dome）中 AI 的使用方式
- 五角大楼首席技术官表示与 Anthropic 在自主战争方面存在分歧
- Anthropic 放弃了约 **2 亿美元的合同**
- 与此同时，OpenAI 与国防部达成了类似协议

**影响**：⚠️ **高**
- 标志着 AI 伦理立场与政府合同之间的直接冲突
- Anthropic 可能损失大量政府收入
- 但也可能强化其"负责任 AI"品牌形象，吸引企业客户

**来源**：[AP News](https://apnews.com/article/ai-anthropic-pentagon-golden-dome-autonomous-weapons-6f3c45ff46172c1bf8658dea0098f3fe) | [The Atlantic](https://www.theatlantic.com/ideas/2026/03/anthropic-pentagon-contract-openai/686285/) | [SecurityWeek](https://www.securityweek.com/pentagons-chief-tech-officer-says-he-clashed-with-ai-company-anthropic-over-autonomous-warfare/)
