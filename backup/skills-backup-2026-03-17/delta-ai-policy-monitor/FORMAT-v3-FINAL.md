## 🎨 排版规范（v3.0）

### 1. 报告标题
```
📊 **AI 政策推手监控 - 日报**
**2026-03-06 | 过去 24 小时**
```

### 2. 大板块标题
**【中国部分】** / **【美国部分】**（不加粗）

### 3. 子板块标题
**政策动态** / **资金支持** / **产业园区** / **Trump 相关**（加粗）

### 4. 新闻格式（每个新闻是独立语义块）

**结构**：
```
**- 新闻标题**（加粗）

　　**简要解析**：内容（缩进+加粗）

　　📎 **来源**：[链接名称](URL)（缩进+加粗）
```

**要点**：
- ✅ 新闻标题加粗
- ✅ 简要解析和来源缩进（两个全角空格 `　　`）
- ✅ 简要解析和来源加粗
- ✅ 每个新闻之间有空行

### 5. 空行规则

**语义块之间空行**：
- 不同新闻之间：空行
- 不同子板块之间：空行
- 中美分隔：横线上下各加空行

**示例**：
```
**- 新闻1**
　　**简要解析**...
　　📎 **来源**...

　

**- 新闻2**
　　**简要解析**...

**资金支持**
```

