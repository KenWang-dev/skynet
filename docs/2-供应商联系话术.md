# 2. 供应商联系话术

## 场景分类

根据你的实际情况选择对应话术：
- **场景A**：已有公司/项目（正式采购）
- **场景B**：个人创业/初期调研（早期阶段）
- **场景C**：代理商/中间商（代采购）

---

## 场景A：已有公司/项目（正式采购）

### 电话话术（首通电话）

**开场（30秒）**：
```
您好，请问是紫建电子的投资者关系部吗？

我是[你的公司名称]的采购经理[你的姓名]。
我们在开发一款智能戒指产品，
了解到贵司在智能戒指电池领域有成熟的合作经验，
想咨询一下电池采购合作的事宜。

请问可以转到业务部门或电池销售部门吗？
```

**如果转接到业务部门**：
```
您好，我是[公司名称]的采购经理[姓名]。
我们正在开发智能戒指产品，
想了解贵司的可穿戴设备电池方案。

我们目前的需求是：
- 电池容量：15-30mAh
- 形状：环形或弧形
- 应用场景：智能戒指
- 采购量：初期X千个/月，量产阶段X万个/月

请问贵司是否有类似的产品案例？
是否可以提供技术规格书和样品？
```

**关键问题清单**（准备问对方）：

| 问题类型 | 具体问题 |
|----------|----------|
| **产品确认** | 有没有智能戒指电池的成功案例？最主流的产品方案是什么？ |
| **技术参数** | 标准产品的容量、尺寸、循环寿命是多少？ |
| **定制能力** | 是否支持定制尺寸和形状？定制周期多久？ |
| **MOQ** | 最小起订量是多少？阶梯定价如何？ |
| **样品** | 是否可以提供样品？样品费用和交付周期？ |
| **认证** | 产品通过了哪些认证（CE/FCC/PSE/UN38.3）？ |
| **交期** | 样品交期多久？量产交期多久？ |
| **价格** | 目标单价区间是多少？量大是否有折扣？ |
| **付款** | 付款方式是什么（月结/预付款）？ |
| **质保** | 质保期多久？质量问题如何处理？ |

### 邮件话术（正式邮件）

**邮件主题**：
```
智能戒指电池采购咨询 - [你的公司名称]
```

**邮件正文**：
```
紫建电子 销售总监/业务团队：

您好！

我是[公司名称]的采购经理[姓名]。
我们公司专注于[公司主营业务，如：可穿戴设备/智能硬件]的研发和销售。

目前我们正在开发一款智能戒指产品，需要采购小型锂离子电池。
通过公开信息了解到贵司在智能戒指电池领域与国际知名品牌有成功合作经验，
希望能够建立合作关系。

我们的初步需求如下：

【产品需求】
- 应用场景：智能戒指（可穿戴设备）
- 电池类型：锂聚合物电池（软包电池）
- 电池容量：15-30mAh（可协商）
- 形状要求：环形或弧形（需定制）
- 续航目标：5-7天
- 安全认证：CE、FCC、PSE等

【采购计划】
- 样品阶段：X个（用于测试验证）
- 小批量试产：X千个/月
- 量产阶段：X万个/月

【咨询事项】
1. 贵司是否有智能戒指电池的成功案例？可否分享参考方案？
2. 是否支持定制化开发？开发周期多久？
3. 是否可以提供技术规格书和样品？
4. MOQ（最小起订量）和阶梯定价如何？
5. 样品费用、交付周期、量产交期分别多久？
6. 产品通过了哪些认证？是否支持做新认证？

随信附上我司简介和项目概况（如有）。

期待您的回复，希望能有机会深入交流。

微信：[微信号]
地址：[公司地址]
```

**附件准备**：

## 场景B：个人创业/初期调研（早期阶段）

### 电话话术

**开场（诚实说明）**：
```
您好，我是[姓名]，是一名创业者/独立开发者。

我目前正在筹备一个智能戒指项目，
现在处于前期调研阶段，
想了解一下贵司的电池产品和技术方案。

虽然目前采购量不大，
但如果项目成功，未来会有持续的采购需求。

请问可以给我介绍一下贵司的智能戒指电池方案吗？
或者可以发一些技术资料给我参考吗？
```

**对方可能会问的问题（提前准备）**：

| 对方问题 | 建议回答 |
|----------|----------|
| "你们公司什么时候成立？" | "目前是个人的创业项目，正在筹备公司注册。" |
| "预期采购量是多少？" | "初期可能几十到几百个，如果产品成功，目标X万个/月。" |
| "有融资吗？" | "正在接触投资人/已获得种子轮投资（如实说）。" |
| "需要什么时候要样品？" | "1-2个月内，想先做技术验证。" |

**关键策略**：
- ✅ 诚实但自信
- ✅ 强调未来潜力
- ✅ 不要过度承诺
- ✅ 表现出专业性（做过功课）

## 场景C：代理商/中间商（代采购）

### 电话话术

**开场**：
```
您好，我是[代理商名称]的业务经理[姓名]。

我们公司为海外/国内客户提供电子元器件采购服务，
目前有一个客户需要采购智能戒指电池，
了解到贵司是这个领域的专业厂商，
想了解一下代理合作或直接采购的政策。

请问贵司是否有代理体系？或直接与终端客户合作？
```

**关键信息**：
- 说明代理模式
- 客户类型（海外/国内）
- 采购规模和频率
- 代理合作意向

## 邮件后续跟进话术

### 跟进时机

- **首封邮件后3天无回复** → 发跟进邮件
- **样品收到后** → 反馈测试结果
- **价格谈判阶段** → 请求折扣或更好的条款
- **项目进展更新** → 同步信息，保持关系

## 常见异议应对

### 异议1："我们不接小订单"

**应对话术**：
```
理解理解。不过我们项目分几个阶段：
- 第一阶段：样品验证（X个）
- 第二阶段：小批量试产（X千个）
- 第三阶段：量产（X万个/月）

虽然现在量不大，
但如果前两个阶段合作顺利，
第三阶段会优先考虑贵司。

是否可以破例支持小批量？
或者有其他合作方案吗？
```

---

### 异议2："我们需要先验厂/审核"

**应对话术**：
```
完全可以理解。

我司位于[城市]，
随时欢迎贵司前来验厂。

同时，我也可以先提供：
- 公司营业执照
- 公司简介和团队介绍
- 项目商业计划书
- 过往合作案例（如有）

请问审核流程需要多久？
需要准备哪些具体材料？
```

---

### 异议3："价格超出预算"

**应对话术**：
```
感谢您的报价。

我理解贵司产品的品质和价值，
但我们项目的成本预算确实有限。

请问：
1. 如果采购量达到[X个]，是否有折扣空间？
2. 是否有性价比更高的标准产品方案？
3. 对于长期合作，是否有价格优惠政策？
4. 能否分期降价（首单X元，后续降到Y元）？

我们希望找到双方都能接受的合作方案，
期待您的建议。
```

---

### 异议4："需要签署NDA才能提供详细资料"

**应对话术**：
```
没问题，我们可以签署NDA。

请问贵司有标准NDA模板吗？
或者我提供NDA草案？

签署后需要多久才能收到技术资料？

[如果对方没有模板]
我可以提供NDA草案，预计1-2天准备完毕。
请问签署后多久可以收到技术规格书和样品报价？
```

## 联系后的行动计划

### 第一次联系后

| 时间 | 行动 |
|------|------|
| **立即** | 记录联系人姓名、电话、邮箱、部门 |
| **当天** | 发送跟进邮件（总结电话内容） |
| **3天后** | 如无回复，发跟进邮件或再次致电 |
| **1周后** | 评估对方合作意愿，决定是否联系备选供应商 |

### 资料整理

每次联系后记录：
- 对方公司名称、联系人
- 产品信息（型号、参数、价格）
- 合作意愿（高/中/低）
- 下一步行动计划
- 对比其他供应商的优势/劣势
