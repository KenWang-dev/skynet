### 步骤 2：修改任务脚本
在报告生成后添加：
```bash
# 加载飞书上传函数
source /root/.openclaw/workspace/scripts/feishu-uploader.sh
source /root/.openclaw/workspace/scripts/feishu-config.sh

# 上传到飞书

# 清理旧文件（超过30天）
```
