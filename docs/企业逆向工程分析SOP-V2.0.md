# 企业逆向工程分析 SOP — Agent 执行手册 V2.0

## 0. 执行总则

**目的**：为 AI Agent 提供一套标准化的企业信息采集与分析流程，用于持续监控指定公司的关键变量。

**核心原则**：
- **第一性原理**：每个分析项追问"底层驱动因素是什么"，不停留在表象数据
- **MECE 法则**：各模块互斥且穷尽，不重复、不遗漏
- **奥卡姆剃刀**：只采集对决策有因果影响的信息，剔除冗余

**执行顺序**：严格按 `定位 → 画布 → 文化 → 生命周期阶段 → 原则提炼` 顺序执行。分析终点由企业当前所处阶段决定。

---

## 1. 定位：判断企业生命周期阶段

> **前置动作**：在深入分析前，先确定目标企业处于哪个阶段，决定后续分析的范围。

| 阶段 | 判定特征 | 分析范围 |
|------|----------|----------|
| **初创期（0→1）** | 产品未成熟、商业模式验证中、营收极小或为零、依赖融资 | 画布 + 文化 + 初创期 |
| **成长期（1→N）** | PMF已验证、用户/营收爆发增长、组织快速扩张、亏损或微利 | 画布 + 文化 + 初创期回顾 + 成长期 |
| **成熟期（N→N+1）** | 增长放缓、市场份额稳定、现金流充裕、面临大公司病 | 画布 + 文化 + 历史回顾 + 成熟期（重点） |
| **衰退期** | 核心业务营收/利润持续下滑、份额被侵蚀、转型压力大 | 全框架完整执行 |

**输出**：`[公司名称] 当前处于 [阶段]，分析范围为 [模块列表]`

## 2. 时代画布：外部约束与可能性边界

### 2.1 宏观环境（PESTEL）

对每个维度，只需回答一个核心问题并给出 **利好 / 中性 / 利空** 的判定。

| 维度 | 核心采集问题 | 关键数据源 |
|------|-------------|-----------|
| **政治** | 政府对该行业的态度（扶持/限制/中立）？关键监管政策方向？贸易政策影响？ | 政策文件、监管动态、贸易协定 |
| **经济** | 所处经济周期阶段？利率/汇率/通胀对该企业的成本与需求影响方向？ | 央行报告、CPI/GDP数据、行业景气指数 |
| **社会** | 目标客群的人口结构变化趋势？消费观念与文化潮流的转向？ | 人口普查、消费者调研、社会趋势报告 |
| **技术** | 影响该行业的颠覆性技术是什么？核心技术处于成熟度曲线的哪个位置？ | 技术报告、专利数据、行业研报 |
| **环境** | 环保政策（碳中和、排放标准）对成本结构的影响？ESG要求趋势？ | 环保法规、ESG评级、行业合规要求 |
| **法律** | 劳动法/知识产权/行业准入的关键约束与变化？ | 法律法规更新、司法案例、合规公告 |

**输出**：6维度判定表 + 综合结论（该企业的宏观环境总体 利好/中性/利空）

### 2.2 产业环境（波特五力）

对每个力量，判定其强度：**强 / 中 / 弱**。

| 力量 | 核心判定逻辑 | 关键指标 |
|------|-------------|---------|
| **供应商议价力** | 供应商集中度高？替代品少？转换成本高？→ 强 | 供应商CR5、原材料替代性、合同锁定期 |
| **买方议价力** | 客户集中度高？产品同质化？转换成本低？→ 强 | 客户CR5、产品差异化程度、客户流失率 |
| **新进入者威胁** | 资本/技术/规模/牌照门槛低？→ 威胁大 | 行业平均初始投资、专利壁垒、规模经济拐点 |
| **替代品威胁** | 存在性价比更优的跨行业替代方案？转换成本低？→ 威胁大 | 替代品渗透率、价格性能比、用户迁移数据 |
| **行业内竞争** | 玩家多？增长慢？产品同质？→ 竞争激烈 | 行业CR10、增长率、价格战频率 |

**输出**：五力强度矩阵 + 行业吸引力综合判定（高/中/低）

### 2.3 赛道研判（熔断机制）

**核心判定**：该行业是否处于向上（或至少不向下）的趋势？
- **YES** → 继续后续分析
- **NO**（行业在萎缩且企业无颠覆性创新）→ 标记为高风险，后续分析价值有限

**输出**：`赛道判定：[YES/NO]，理由：[一句话]`

## 3. 文化内核：组织的操作系统

> 文化决定企业行为模式的底层逻辑，需在分析任何具体业务前完成。

### 3.1 创始人精神印记

| 分析项 | 采集内容 |
|--------|---------|
| **价值观与世界观** | 核心信仰是什么？技术信仰/市场信仰/用户信仰？对人性的基本假设？ |
| **决策元算法** | 数据驱动 vs 直觉判断？完美主义 vs 快速迭代？长线思维 vs 短线出击？ |
| **关键经历** | 哪些成败经历塑造了其行为模式和决策禁忌？ |
| **高压决策质量** | 在公司生死时刻，是否做出过反直觉但符合第一性原理的关键决策？是否有"非共识做对"的记录？ |

### 3.2 核心价值主张

| 分析项 | 采集内容 |
|--------|---------|
| **宣称 vs 实践** | 公开价值观是什么？在利益冲突时是否被坚守？ |
| **行为标尺** | 什么行为被最高级别奖励？什么行为即便带来短期业绩也会被惩罚？ |
| **优先级排序** | 当价值观冲突时（如创新 vs 成本控制），哪个胜出？ |

### 3.3 激励系统

| 分析项 | 采集内容 |
|--------|---------|
| **奖励导向** | KPI/OKR 奖励的是执行者、英雄、创新者还是协作者？ |
| **容错机制** | 对"有才无德"的态度？对创新失败的容忍度？ |
| **公平性** | 规则是否清晰透明？组织凝聚力还是离心力？ |

### 3.4 组织叙事

| 分析项 | 采集内容 |
|--------|---------|
| **创始神话** | 内部流传的创业故事是什么？传递什么精神？ |
| **英雄人物** | 谁被视为内部榜样？因何被铭记？ |
| **文化符号** | 是否有独特的内部术语、行话、仪式？ |

**输出**：文化画像摘要（200字内概括该企业的文化操作系统特征）

## 4. 生命周期阶段分析

> 根据第1步的定位结果，执行对应阶段的分析模块。

### 4.1 初创期（0→1）：产品-市场验证

**核心命题**：是否找到了 PMF（Product-Market Fit）？

#### A. 根基层

| 分析项 | 采集内容 |
|--------|---------|
| **团队能力圈** | 核心团队是否能力互补？关键短板在哪？ |
| **认知与愿景** | 对行业终局的判断？使命与愿景？ |
| **股权结构** | 是否合理反映贡献？决策机制是否清晰？是否预留期权池？ |
| **资金来源** | 自有资金 vs 风险投资？对应的增长压力？ |
| **技术来源** | 自研独创 vs 开源应用？初期技术壁垒高低？ |
| **关系网络** | 是否有带来首批客户/人才/洞见的非对称优势？ |

#### B. 执行层

| 分析项 | 采集内容 |
|--------|---------|
| **价值主张** | 为谁解决什么痛点，创造什么价值？（一句话） |
| **MVP 策略** | 最小可行产品的功能边界？是否聚焦核心验证？ |
| **收入模式** | 直接销售/订阅/广告/交易抽成？ |
| **成本结构** | 最大开销项（人力/研发/营销）？ |
| **冷启动策略** | 如何获取种子用户？社区/内容/合作伙伴？ |
| **早期招聘标准** | 前20人的选人逻辑？能力 vs 愿景认同的权重？ |

#### C. 增长飞轮

- **飞轮组件**：构成增长循环的核心动作是什么？
- **反馈循环**：Build → Measure → Learn 循环是否形成？
- **飞轮动能**：当前转速？最大摩擦力来源？

---

### 4.2 成长期（1→N）：规模化与组织进化

**核心命题**：已验证的模式能否被大规模高效复制？

#### A. 根基层

**核心竞争力（VRIO 检验）**：

| 检验维度 | 判定问题 | 结论 |
|----------|---------|------|
| **V — 价值** | 该能力能抓住机会或抵御威胁吗？ | 是/否 |
| **R — 稀有** | 多少竞争者拥有同等能力？ | 稀有/常见 |
| **I — 难模仿** | 复制成本多高？（历史路径/因果复杂/法律保护） | 高/中/低 |
| **O — 组织化** | 组织架构和流程能充分发挥该能力吗？ | 是/否 |

**融资历程**：

| 采集项 | 内容 |
|--------|------|
| 融资节奏与估值曲线 | 各轮次时间、金额、估值变化 |
| 投资方性质 | 财务投资 vs 战略投资？ |
| 战略资源注入 | 投资方带来的客户/人才/渠道等非资金价值 |

#### B. 执行层

**关键里程碑**：

| 采集项 | 内容 |
|--------|------|
| 产品迭代路径 | MVP → 成熟产品线的演化逻辑？拓新客 vs 锁老客？ |
| 市场扩张路径 | 种子市场 → 规模市场的策略？直销/渠道/生态？地域逻辑？ |
| 关键战役复盘 | 1-2次关键竞争事件的过程、结果与暴露的优劣势 |

**组织诊断（7S 检查清单）**：

| S维度 | 核心诊断问题 |
|-------|-------------|
| **战略** | 战略能否有效传达到基层？是否存在信息衰减？ |
| **结构** | 从扁平化到职能/事业部的转型是否完成？部门墙问题？ |
| **制度** | 核心流程（研发/销售/财务）是否标准化？还是靠创始人拍板？ |
| **人员** | 人才密度被稀释了吗？靠空降兵还是自建培养体系？ |
| **技能** | 关键能力短板（数据/项目管理/……）？ |
| **风格** | 领导从一线指挥官转变为组织设计师了吗？ |
| **共同价值观** | 文化内核在新人涌入后是否被有效传承？ |

#### C. 增长飞轮

- **飞轮演化**：相比初创期，是否加入新组件？
- 网络效应？规模效应？品牌效应？数据智能？
- **反馈循环**：新飞轮是否具备更强的自加速特性？
- **飞轮动能**：当前加速力（资本/技术/营销）？当前摩擦力（竞争/组织效率/技术架构）？

---

### 4.3 成熟期（N→N+1）：守成与创新的平衡

**核心命题**：第一曲线能否守住？第二曲线能否接棒？

#### A. 第一曲线巩固

**业务组合诊断（BCG 矩阵）**：

| 象限 | 定义 | 策略 |
|------|------|------|
| 现金牛 | 低增长 + 高份额 | 收获现金，供血其他业务 |
| 明星 | 高增长 + 高份额 | 持续投入，培育为下一个现金牛 |
| 问号 | 高增长 + 低份额 | 审慎决策：加注或止损 |
| 瘦狗 | 低增长 + 低份额 | 收缩或剥离 |

**财务健康（杜邦分解）**：

| 指标 | 采集内容 |
|------|---------|
| ROE | 整体盈利能力 |
| 驱动归因 | 高利润率（品牌/技术溢价）？高周转率（运营效率）？高杠杆（债务风险）？ |

**护城河分析**：

| 采集项 | 内容 |
|--------|------|
| 护城河类型 | 无形资产/转换成本/网络效应/成本优势？ |
| 护城河深度 | 竞争者突破需要多大代价？ |
| 侵蚀风险 | 技术变革/政策变化/竞争者是否正在削弱护城河？ |
| 加固措施 | 企业在做什么来加深护城河？ |

**组织效能**：

| 采集项 | 内容 |
|--------|------|
| 官僚化程度 | 决策流程冗长？部门墙？创新被流程扼杀？ |
| 人才密度 | 能否吸引和留住顶尖人才？还是优秀人才在流失？ |

#### B. 第二曲线分析

| 分析项 | 采集内容 |
|--------|---------|
| **候选业务** | 公司将哪些新业务视为未来增长引擎？（依据：研发投入、并购、高管发言） |
| **战略意图** | 主动预判布局 vs 被动防御跟随？ |
| **发展阶段** | 第二曲线自身处于 0→1 还是 1→N？ |
| **核心瓶颈** | 技术未突破？商业模式未跑通？缺组织和人才？ |
| **供血机制** | 第一曲线向第二曲线的资源分配是否稳定持续？ |
| **协同效应** | 第二曲线能否复用第一曲线的品牌/渠道/技术/数据？ |
| **创新者窘境** | 成熟业务部门是否在打压新业务？组织架构如何化解冲突？ |
| **接棒路线图** | 何时、如何让第二曲线从边缘走向中心？是否有清晰规划？ |

#### C. 双飞轮分析

- 第一曲线飞轮是否减速？摩擦力来源？
- 第二曲线飞轮是否已形成？是否已展现自加速潜力？
- 两个飞轮是否存在协同（用户/数据/技术互通）？

---

### 4.4 衰退期：转型或退出

**核心命题**：能否自救？如何最大化残余价值？

#### A. 诊断层

**衰退表征**：

| 信号 | 关键指标 |
|------|---------|
| 财务恶化 | 连续N个季度营收下滑、利润率萎缩、现金流枯竭 |
| 份额流失 | 核心产品被竞品/替代品蚕食 |
| 人才外流 | 核心人才用脚投票 |
| 创新停滞 | 多年无有竞争力的新产品 |

**根因分析**（互斥归因）：

| 类型 | 判定 |
|------|------|
| 行业性衰退 | 整个行业因技术/社会变迁走向消亡？ |
| 外部颠覆 | 行业外进入者用新技术降维打击？ |
| 自身战略失误 | 固守旧技术/错失转型/灾难性并购？ |
| 文化僵化 | 曾经的成功文化基因变成了扼杀创新的癌症？ |

**第二曲线复盘**：成熟期布局的第二曲线为何失败？未找到PMF？竞争失败？被内部扼杀？

#### B. 执行层

| 策略类型 | 核心动作 | 关键指标 |
|----------|---------|---------|
| **收缩** | 大规模削减成本、裁员、剥离瘦狗业务 | 成本降幅、业务剥离清单 |
| **转型** | 利用残存核心能力进入新市场或重塑商业模式 | 新业务营收占比、PMF信号 |
| **退出** | 停止长期投资、榨取剩余现金流、出售资产 | 资产处置进度、股东回报 |

## 5. 原则提炼：从案例到可迁移智慧

> 每次分析结束后，必须完成以下三个提炼。

| 提炼维度 | 核心追问 |
|----------|---------|
| **第一性原理** | 在决定公司命运的十字路口，决策背后遵循的根本原则是什么？ |
| **非共识的正确** | 哪些当时不被理解甚至被嘲笑、但事后证明构建了核心壁垒的关键决策？ |
| **穿越周期的能力** | 什么要素让企业穿越经济周期和技术浪潮？文化韧性？反脆弱架构？现金流管理？ |

**输出**：3-5条可迁移的行动原则。

## 6. Agent 监控执行规范

### 6.1 信息采集频率

| 信息类型 | 频率 | 来源 |
|----------|------|------|
| 财务数据（营收/利润/现金流） | 季度 | 财报、SEC/交易所公告 |
| 竞争格局变化 | 月度 | 行业研报、新闻、竞品动态 |
| 管理层动态（人事/战略发言） | 实时 | 公告、采访、会议纪要 |
| 政策与法规变化 | 实时 | 政府文件、监管公告 |
| 技术变革信号 | 月度 | 专利、论文、技术发布 |
| 产品迭代与市场反馈 | 双周 | 产品更新日志、用户评价、应用商店数据 |

### 6.2 预警触发条件

以下任一条件触发时，Agent 应立即生成预警报告：

- 季度营收同比下滑 > 10%
- 核心高管离职（CxO 级别）
- 行业重大政策变化（直接影响该企业）
- 颠覆性竞品发布或新进入者获大额融资
- 核心客户/供应商重大变动
- 第二曲线业务出现 PMF 信号或重大失败

### 6.3 输出格式规范

每次分析输出统一为以下结构：

```
# [公司名称] 分析报告 — [日期]

## 0. 定位判定
[生命周期阶段 + 分析范围]

## 1. 时代画布
[PESTEL 6维判定表 + 五力矩阵 + 赛道研判结论]

## 2. 文化画像
[文化操作系统摘要]

## 3. 阶段分析
[根基层 + 执行层 + 飞轮分析]

## 4. 原则提炼
[3-5条可迁移原则]

## 5. 关键风险与机会
[Top 3 风险 + Top 3 机会]

## 6. 监控重点
[下一周期需重点关注的变量]
```
