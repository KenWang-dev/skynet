### 5. 扩展规律固化 ✅

#### 5.1 编码组合规则
**格式**：`[字母系统][数字]`

**字母系统**：
- **AI 类**：希腊字母（Alpha, Beta, Gamma... 24个）
- **采购类**：拉丁字母（A, B, C... Z 26个）
- **未来类别**：可定义新字母系统（如西里尔字母、数字前缀等）

**数字频率**：
- **1** = 日报（Daily）
- **2** = 周报（Weekly）
- **3** = 月报（Monthly）
- **4** = 季报（Quarterly）
- **5** = 半年报（Half-year）
- **6** = 年报（Annual）

#### 5.2 扩展规则
**新增任务（现有类别）**：
1. 选择对应字母系统的下一个可用字母
2. 确定频率（1-6）
3. 组合编码（如：`Delta1`, `J2`）

**新增类别（全新领域）**：
1. 选择新的字母系统（避免冲突）
2. 定义该类别的字母表
3. 从第一个字母开始分配
