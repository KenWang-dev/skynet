# SkyNet天网监控系统：AI驱动的智能情报系统

## 背景

在信息爆炸时代，决策者面临三大挑战：
- **信息过载**：互联网信息太多，无法筛选关键内容
- **时效性差**：手动搜索信息，无法及时获取最新动态
- **碎片化严重**：信息分散在各个平台，无法形成系统认知

**问题**：如何用AI重构情报获取方式，实现自动化、系统化、智能化的决策支持？

---

## 解决方案

### 系统架构

SkyNet把互联网作为巨大的情报来源，通过结构化监控和多维度分析，为决策提供全面、准确、及时的信息支持。

**双轨并行**：
- **AI类监控**（7个视角）：技术、标杆、资本、政策、人才、应用、社会影响
- **采购类监控**（12个任务）：供应链、风险、政策、财务、市场、实践、心声、ESG、供应商

### 设计方法论

**第一性原理**：
智能决策 = 信息 + 分析 + 预测 + 验证

**MECE法则**（相互独立、完全穷尽）：
- AI生态系统：供给（技术、人才、资本）+ 需求（应用）+ 影响（社会）
- 采购决策：供应链、风险、政策、财务、市场等9个维度

**帕累托法则**（80/20）：
- 30%的核心任务 → 85%的决策价值
- Alpha（技术）、Gamma（资本）、Delta（政策）为TOP3
- A（供应链）、B（风险）、C（政策）、D（财务）为TOP4

### 监控维度

**AI类（7个视角）**：
1. **Alpha（技术前沿）**：技术往哪走？（Karpathy AI博客精选）
2. **Beta（行业标杆）**：巨头怎么做？（AI三巨头监控）
3. **Gamma（资本流向）**：钱往哪流？（6家顶级风投监控）
4. **Delta（政策走向）**：国家推手往哪走？（中美政府政策监控）
5. **Zeta（人才流动）**：人才往哪去？（AI岗位、薪资、热门技能监控）
6. **Eta（应用落地）**：应用往哪落地？（AI应用ROI监控）
7. **Theta（社会影响）**：社会往哪变化？（尤瓦尔·赫拉利历史视角）

**采购类（12个任务）**：
- **活下去（生存）**：A（电子供应链）、B（风险）、C（政策）、D（财务）
- **活得好（发展）**：F（最佳实践）
- **活得久（可持续）**：E（市场）、G（心声）、H（ESG）、I（供应商生态）

---

## 技术实现

### 100%自动化流程

```
监控触发（Cron定时）
  ↓
数据收集（Tavily搜索API）
  ↓
智能分析（AI Agent自动生成）
  ↓
报告发送（飞书API推送）
  ↓
用户阅读（早晨7-9点自动送达）
```

### 技术栈

- **搜索引擎**：Tavily Search API（AI优化）
- **AI引擎**：GLM-4.7（智谱AI）
- **定时任务**：OpenClaw Cron系统
- **通讯渠道**：飞书（自动推送）
- **报告格式**：Markdown（结构化输出）
