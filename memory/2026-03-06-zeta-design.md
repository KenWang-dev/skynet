## 🎯 核心逻辑

### 为什么需要人才视角？

**前四个视角的完整性**：
1. **技术视角（Alpha）**：技术往哪走？
2. **标杆视角（Beta）**：巨头怎么做？
3. **资本视角（Gamma）**：钱往哪流？（民间资本）
4. **政府视角（Delta）**：国家推手往哪走？（官方力量）

**第五个视角（Zeta）**：
- **人才往哪去？**
- **验证**：企业是否真的拿到了资金并开始商业落地？
- **落地**：给政策、给扶持、给基金、有技术，但人才来不来？

---

## 📊 监控维度

### 1. 岗位数量（增量监控）

**纵向对比**（时间维度）：
- AI相关岗位的数量变化
- 与过去对比（环比、同比）
- 增长趋势

**横向对比**（岗位维度）：
- AI岗位 vs 其他岗位
- AI岗位占比
- 如果其他岗位都没有/很少，而AI岗位一大堆 = 需求非常大

**数据来源**：
- LinkedIn（领英）
- BOSS直聘
- 脉脉
- 猎聘
- 拉勾网

### 2. 薪资水平（真金白银）

**中位数薪资**：
- AI岗位薪资中位数
- 与其他岗位对比
- 溢价水平（20%？50%？一倍？）

**薪资分布**：
- 不同岗位的薪资区间
- 薪资趋势
- 真金白银的需求

**数据来源**：
- LinkedIn
- BOSS直聘
- 脉脉
- 猎聘

### 3. 热门岗位（市场需求）

**新增岗位**：
- 以前没有的岗位
- 新兴岗位
- 创新的岗位名称

**需求量大的岗位**：
- 每个公司都招的岗位
- 高频出现的岗位
- 热门岗位名称

**热门岗位示例**（推测）：
- AI算法工程师
- 大模型开发工程师
- Prompt工程师
- AI产品经理
- AIGC内容运营
- AI解决方案架构师

**数据来源**：
- LinkedIn
- BOSS直聘
- 脉脉
- 小红书
- Reddit
- Blind

### 4. 地域分布（人才流向）

**热门城市**：
- AI岗位最多的城市
- 新一线城市 vs 一线城市
- 人才流向趋势

**产业集群**：
- 哪些地区AI岗位集中
- 产业集群 vs 政策扶持
- 地域人才竞争

**数据来源**：
- LinkedIn
- BOSS直聘
- 脉脉
- 各地人才政策

### 5. 企业需求（商业落地验证）

**企业类型**：
- 大厂 vs 创业公司
- 知名企业 vs 新锐企业
- 外企 vs 国企 vs 民企

**岗位要求**：
- 技能要求
- 经验要求
- 学历要求
- 薪资范围

**验证逻辑**：
- 如果企业大规模招聘AI人才 = 真的拿到资金 + 开始商业落地
- 真金白银的人才需求 = 最真实的商业信号

**数据来源**：
- LinkedIn
- BOSS直聘
- 脉脉
- 企业官网
- 招聘网站

## 📋 监控任务设计

### Zeta1 - AI人才流动监控（日报）

**时间**：每天 8:45
**目的**：快速扫描过去 24 小时的AI人才流动

**监控内容**：
1. **岗位数量**：
   - AI岗位新增数量
   - 环比变化
   - 与其他岗位对比

2. **薪资水平**：
   - AI岗位薪资中位数
   - 与其他岗位对比
   - 薪资溢价

3. **热门岗位**：
   - 新增岗位
   - 需求量大的岗位
   - 热门岗位名称

4. **企业动态**：
   - 大厂招聘动态
   - 创业公司招聘
   - 地域分布

**报告格式**：汇报风格（REPORT_STYLE_STANDARD.md）

### Zeta2 - AI人才流动监控（周报）

**时间**：每周六 8:48
**目的**：深度分析过去 7 天的AI人才流动趋势

**监控内容**：
1. **趋势分析**：
   - 岗位数量趋势（7天变化）
   - 薪资水平趋势
   - 热门岗位变化

2. **地域分布**：
   - 热门城市排名
   - 人才流向趋势
   - 产业集群分析

3. **企业需求**：
   - 大厂 vs 创业公司
   - 行业分布
   - 技能要求变化

4. **商业落地验证**：
   - 哪些企业在大规模招聘？
   - 招聘规模如何？
   - 薪资水平如何？
   - 是否真的拿到资金并开始商业落地？

5. **交叉验证**：
   - 与 Alpha（技术视角）：热门技能 vs 技术前沿
   - 与 Beta（标杆视角）：企业需求 vs 巨头动态
   - 与 Gamma（资本视角）：招聘规模 vs 融资动态
   - 与 Delta（政府视角）：地域分布 vs 政策扶持

**报告格式**：汇报风格（REPORT_STYLE_STANDARD.md）

## 🎯 核心价值

### 1. 验证商业落地
**问题**：企业融到资了吗？开始商业化了吗？
**答案**：看招聘！大规模招聘 = 真金白银的商业落地

### 2. 真实需求
**问题**：AI需求是真实的还是炒作？
**答案**：看岗位数量和薪资！真金白银的招聘 = 真实需求

### 3. 人才流向
**问题**：人才往哪里去？
**答案**：看热门城市、热门企业、热门岗位

### 4. 技能趋势
**问题**：市场需要什么技能？
**答案**：看岗位要求和薪资水平

### 5. 职业指导
**问题**：应该学什么？去哪里？
**答案**：看热门岗位和薪资水平，提供职业建议
