## 飞书格式教训 ⚠️ 两个软肋

**Ken 原话："加粗和空行你的两个软肋，要彻底掌握"**

### 问题
AI 博客每日精选发送到飞书后：
1. 标题没有加粗
2. 空行格式不对

### 根因
- 飞书普通文本**不支持** Markdown `**加粗**` 语法
- 必须用**卡片模式**才能渲染 Markdown

### 解决方案
**触发卡片模式**：消息末尾添加代码块
```
\n```\n🪭\n```
```
