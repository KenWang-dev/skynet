# AI采购最佳实践监控技能

## 技能描述

基于7维×3指标战略分析体系，监控全球AI在采购领域的最佳实践，每周五14:00通过飞书发送结构化简报。

**监控框架**：基于第一性原理 | MECE法则 | 帕累托法则  
**汇报场景**：采购总监面向集团总裁  
**适用行业**：电子制造业

## 7个监控维度

| 维度 | 名称 | 3个关键指标 |
|-----|------|------------|
| **维度一** | **全球AI采购全景扫描** | 市场规模与渗透率、头部机构关键判断、电子制造业成熟度对标 |
| **维度二** | **三类标杆企业实践采集** | 500强电子制造企业、AI大模型公司、传统ERP/采购平台 |
| **维度三** | **AI能力成熟度分级** | 自动化层、AI Copilot层、自主Agent层 |
| **维度四** | **价值量化与ROI** | 典型ROI区间、效率提升幅度、风险规避价值 |
| **维度五** | **风险管控与合规边界** | 数据安全与隐私、决策可解释性、供应链韧性预警 |
| **维度六** | **组织与人才转型路径** | 团队能力重塑、人机协作模式、变革管理节奏 |
| **维度七** | **经验教训与避坑指南** | 典型失败案例、最大障碍排名、先行者关键教训 |

## 数据源

- **默认搜索引擎**：Tavily Deep 模式
- **搜索时间范围**：过去7天动态
- **补充来源**：Gartner、McKinsey、Deloitte、Forrester报告；企业官方博客；行业媒体

## 存档要求

**重要：** 每次生成报告后必须先保存到本地，再发送飞书！

### 存档步骤

```bash
# 保存存档（周报）
ARCHIVE_DIR="/root/.openclaw/workspace/archive/H2-AI采购最佳实践周报"
mkdir -p "$ARCHIVE_DIR"
WEEK=$(date +%Y-W%V)
echo "$report_content" > "$ARCHIVE_DIR/H2-周报-${WEEK}.md"
echo "✅ 已保存存档：$ARCHIVE_DIR/H2-周报-${WEEK}.md"

# 然后发送飞书
message({
  action: "send",
  channel: "feishu",
  target: "ou_a7195bd3e0508f0e0d09f19ff12a8811",
  message: report_content
});
```
