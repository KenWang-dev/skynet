## 🔍 核心原则

**先收集，后归纳，再分类**

严禁"定向证据采集"——即先预设结论再找支撑素材。

正确流程：
1. ✅ 广泛采集原始语料（不带预判地大量收集）
2. ✅ 对语料做自然聚类（让主题从数据中涌现）
3. ✅ 将涌现出的主题归入分析框架（用框架定位，不用框架预设）
4. ✅ 统计各维度/指标的声量分布与情绪浓度
5. ✅ 基于数据输出洞察，对接培训设计
