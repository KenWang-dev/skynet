# 全球采购真实心声监控

## 概述

面向全球采购从业者群体，系统性收集其在公开社交平台与职业社区上的真实心声（抱怨、吐槽、求助、宣泄、反思等），通过6维度×3指标分析框架，提炼高频共性问题，为采购培训计划提供数据支撑。

**核心原则**：先收集，后归纳，再分类（严禁"定向证据采集"）

**详细分析框架**：见 `reference/原始任务书.md`

## 监控框架概览

**6个监控维度**：
- **D1 痛点主题**：工作负荷与效率、薪酬与职业发展、专业能力挑战
- **D2 抱怨对象**：供应商端、内部上级/跨部门、制度与系统
- **D3 情绪光谱**：🟡吐槽调侃、🟠焦虑疲惫、🔴崩溃绝望
- **D4 事态阶段**：困局僵持中、妥协推进中、已解决/复盘中
- **D5 地域文化**：中国、印度、北美/欧洲
- **D6 角色画像**：执行层（0-3年）、中层管理（3-8年）、高层决策（10年+）

> 📖 **详细分析框架**见 `reference/原始任务书.md`

## 监控流程

### 步骤 1：数据采集

使用 **Tavily Deep 搜索**（过去7天）收集各平台采购从业者心声：

```bash
bash /root/.openclaw/workspace/skills/global-procurement-voice-monitor/run.sh
```

**搜索维度**：16个并行搜索（每个维度最多15条结果）
- 覆盖平台：LinkedIn、Reddit、知乎、小红书、脉脉、CIPS、ISM、Glassdoor
- 关键词策略：职业身份词 + 情绪/行为信号词组合

### 步骤 2：语料分类

按照 [原始任务书](reference/原始任务书.md) 中的6维度框架对收集的心声进行标注和归类。

### 步骤 3：生成周报

#### 报告结构

**顶部：采集概览**
- 采集总量、覆盖平台、涉及地区

**中部：维度分布统计**
- D1-D6 各维度指标占比和趋势

**核心部分：TOP 5高频话题**
- 提炼最常出现的话题
- 包含代表性原声摘录

**底部：培训需求信号**
- P0/P1/P2 优先级排序
- 对应目标群体和地域

#### 输出格式要求

- **语言**：简体中文
- **排版**：清晰易读，关键数据加粗，使用emoji标注情绪级别
- **原则**：数据驱动，有原始语料支撑，不编造推断
- **核心**：先收集后归纳，严禁"定向证据采集"
