## 🎯 编号规则

| 字段 | 说明 |
|------|------|
| **编号** | 字母(A-P) + 数字(1-3) |
| **字母** | 任务类型 |
| **数字** | 频率（1=日报，2=周报，3=月报） |
| **类型** | AI / 采购 |
| **层次** | 生存 / 发展 / 可持续（仅采购类） |
| **时间** | 执行时间 |
| **频率** | 每天 / 周日 / 每月1日 |
