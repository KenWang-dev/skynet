# 供应链风险监控系统

**7维度 × 3指标 = 21项核心风控指标**

电子制造业采购总监向集团总裁汇报的外部风险监控工具。

---

## 📊 两种报告模式

| 模式 | 频率 | 时间范围 | 内容 | 阅读时长 |
|------|------|----------|------|----------|
| 🚨 风险日报 | 每天 7:50 | 过去 24h | 只显示黄灯+红灯 | < 2 分钟 |
| 📋 风险周报 | 每周六 10:00 | 过去 7 天 | 全部 21 项指标 | < 10 分钟 |

---

## 工作流程决策树

```
用户请求
├─ "今日风险" / "生成本日风险报告"
│  └─ 风险日报（只搜索黄灯+红灯指标）
│
├─ "本周风险" / "生成本周风险报告"
│  └─ 风险周报（全部 21 项指标）
│
├─ "监控供应链风险" / "设置监控"
│  └─ 配置定时任务（日报+周报）
│
├─ "检查LME铜价" / "AI虹吸效应怎么样？"
│  └─ 单项指标查询（快速搜索）
│
└─ "本月风险综述" / "季度分析"
   └─ 生成深度分析报告（月度/季度）
```

## 执行步骤

### 风险日报（快速预警版）

1. **读取指标状态**：从 `references/baseline-status.md` 获取上次各指标状态
2. **只搜索异常指标**：对黄灯+红灯指标执行 Tavily 搜索
3. **更新风险等级**：基于搜索结果判断颜色变化
4. **生成日报**：只显示异常指标，简洁明了
5. **发送飞书**：直接发送报告内容

### 风险周报（完整监控版）

1. **搜索全部 21 项指标**：使用 Tavily Deep Research 模式
2. **判断风险等级**：对照 `references/thresholds.md` 判断颜色
3. **生成分维度报告**：7 条消息（每个维度一条）
4. **发送飞书**：分批发送，避免内容压缩

## 搜索策略

**时间过滤**：
- 日报：`freshness: "pd"`（过去 1 天）
- 周报：`freshness: "pw"`（过去 7 天）

**关键词策略**：
- 优先英文关键词（全球数据源更全）
- 辅助中文关键词（本地化数据）
- 使用 `references/data-sources.md` 中的预设关键词

---

## 风险等级判断

| 颜色 | 触发条件 | 应对机制 |
|-----|---------|---------|
| 🟢 绿灯 | 在正常范围内 | 纳入周报，持续跟踪 |
| 🟡 黄灯 | 触发预警条件 | 48小时内根因分析 |
| 🔴 红灯 | 触发危机条件 | 立即启动 War Room |

**阈值参考**：`references/thresholds.md`

---

## 格式规范（v3.1）

**通用规范**：
- ✅ 标题加粗：**【执行摘要】**
- ✅ 全角空格分隔：　　（飞书会压缩连续换行）
- ✅ 诚实标注：有数据标注来源，无数据标注"搜索无结果"
- ✅ 不编造、不推断数据

**发送规范**：
- ❌ 禁止发送执行过程、思考步骤、调试信息
- ✅ 只发送最终报告内容（从标题到签名）
- ✅ 结尾使用："明天见！"

**详细报告模板**：见 `references/report-examples.md`

## 21 项指标速查

| 维度 | 指标 | 数据源 |
|------|------|--------|
| **D1 原材料** | LME铜价、CCL价格、PA66价格 | LME、生意社 |
| **D2 金融** | 汇率、LPR、劳动力成本 | 央行、CFETS |
| **D3 技术** | AI虹吸效应、芯片工艺、EDA工具 | TrendForce |
| **D4 地缘** | 贸易政策、制裁清单、区域冲突 | 官方公告 |
| **D5 供应商** | 核心供应商、替代来源、财务健康 | 企业财报 |
| **D6 物流** | 海运成本、咽喉点通行、仓储成本 | SCFI、马士基 |
| **D7 合规** | 环保法规、产品认证、尽调要求 | 官方法规 |

**完整框架**：见 `references/framework.md`

## 定时任务配置

**日报**：每天 7:50
```json
{
  "name": "B1 - 供应链风险日报",
  "schedule": "50 7 * * *",
  "payload": "执行供应链风险日报：只搜索黄灯和红灯指标（过去24小时），快速预警异常风险"
}
```

**周报**：每周六 10:00
```json
{
  "name": "供应链风险监控-周报",
  "schedule": "20 8 * * 6",

}
```
