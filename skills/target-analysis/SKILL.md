# 标的分析框架

## 核心理念

研究任何标的，本质上是回答六个递进的元问题：

| 元问题 | 分析层 | 认知动作 |
|--------|--------|----------|
| 看到了什么？ | ① 现象层 Surface | 感知 |
| 它是什么？ | ② 本体层 Entity | 定义 |
| 谁在参与？ | ③ 角色层 Actors | 映射 |
| 它在哪里？ | ④ 生态层 Ecosystem | 定位 |
| 怎么运转？ | ⑤ 机制层 Mechanism | 解构 |
| 改变了什么？ | ⑥ 影响层 Impact | 评估 |

**设计原则**：
- **标的唯一性**：不按属性分类，将研究对象视为唯一标的
- **第一性原理**：每层指标指向最底层因果逻辑
- **帕累托聚焦**：每层只保留 3 个关键指标（20%分析覆盖80%信息）
- **弹性权重**：不同标的类型，六层权重分布不同

---

## 6×3 分析矩阵速查

### ① 现象层 Surface —— 看到了什么？

| 指标 | 关键追问 |
|------|---------|
| **传播路径** | 最初从哪个平台/圈层引爆？经历了哪些传播节点？自上而下还是自下而上？ |
| **情绪触点** | 触发好奇、焦虑、兴奋、愤怒还是共鸣？情绪钩子是否可复制？ |
| **时间节奏** | 当前处于萌芽期、爆发期、平台期还是衰退期？热度曲线形态？ |

### ② 本体层 Entity —— 它是什么？

| 指标 | 关键追问 |
|------|---------|
| **核心价值主张** | 一句话说清它为什么存在？满足刚需、痒点还是兴奋点？ |
| **差异化特征** | 去掉什么特征它就不再是它？差异化是技术/体验/认知壁垒？ |
| **体验结构** | 用户旅程中哪些节点是关键转化点？哪个环节决定留存或流失？ |

### ③ 角色层 Actors —— 谁在参与？

| 指标 | 关键追问 |
|------|---------|
| **操盘手画像** | 创始人/核心团队背景、动机、决策风格？过去哪些决策塑造了标的走向？ |
| **用户画像** | 最活跃用户是谁？使用场景和真实痛点？表面需求与底层需求是否错位？ |
| **利益相关方** | 投资人、合作伙伴、监管方、媒体、竞对的立场和诉求？谁有能力"杀死"或"催熟"？ |

### ④ 生态层 Ecosystem —— 它在哪里？

| 指标 | 关键追问 |
|------|---------|
| **市场定位** | 市场多大？处于什么发展阶段？天花板是结构性还是阶段性的？ |
| **竞争格局** | 直接竞品和间接替代方案分别是谁？竞争核心维度是价格/体验/技术/生态？ |
| **上下游关系** | 依赖哪些关键资源/平台？谁依赖它？上游断供或下游流失的影响多大？ |

### ⑤ 机制层 Mechanism —— 怎么运转？

| 指标 | 关键追问 |
|------|---------|
| **增长引擎** | PLG/营销驱动/网络效应/内容驱动？飞轮启动条件和加速条件？ |
| **商业模式** | 钱从哪来？价值如何在各参与方之间流转？单位经济模型是否成立？ |
| **能力壁垒** | 护城河是技术专利/数据积累/品牌认知/网络效应/规模经济？护城河在加深还是被侵蚀？ |

### ⑥ 影响层 Impact —— 改变了什么？

| 指标 | 关键追问 |
|------|---------|
| **行业重塑** | 是否改变了行业竞争规则？催生了哪些新角色或淘汰了哪些旧角色？ |
| **社会涟漪** | 改变了人们哪些行为习惯？引发了哪些新社会议题或文化现象？ |
| **演化方向** | 接下来最可能往哪走？终局是被收购/上市/成为基础设施/昙花一现？ |

## 标准研究流程

```
Step 1 → 标的识别：明确研究标的，不做属性分类
Step 2 → 快速扫描：用六层元问题快速扫描，每层用1-2句话回答
Step 3 → 权重判定：确定哪2-3层是该标的的分析重心
Step 4 → 深度分析：对重心层的3个指标逐一深挖，输出结构化分析
Step 5 → 交叉验证：检查各层结论是否逻辑自洽，是否有信息矛盾
Step 6 → 结论输出：综合六层分析，给出核心判断和行动建议
```

**信息收集顺序**（易 → 难）：
现象层 → 本体层 → 角色层 → 生态层 → 机制层 → 影响层

**输出呈现顺序**（逻辑优先）：
本体层 → 现象层 → 角色层 → 生态层 → 机制层 → 影响层

## 质量检查清单

完成分析后自检：
- [ ] 每层的 3 个指标是否都有事实或数据支撑，而非仅凭主观判断？
- [ ] 各层之间的结论是否逻辑一致？（如本体层说差异化很强，但机制层说护城河薄弱——是否矛盾？）
- [ ] 是否遗漏了某个层的关键信息？
- [ ] 分析重心是否选对了？（标的类型判断是否准确？）
- [ ] 最终结论能否用一段话清晰表达？
