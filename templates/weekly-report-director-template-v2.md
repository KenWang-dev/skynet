## 🎨 排版规范

### ⚠️ 关键约束（v2.0 - 2026-03-08 固化）

#### 1. 影响级别排序规则
**核心洞察必须按影响级别从高到低排序：**
- 🔴 高影响（第1-2条）
- 🟡 中影响（第3-4条）
- 🟢 低影响（第5条）

**❌ 禁止**："高中中低高"的混乱排序
**✅ 正确**："高高中中低"或"高中中低"（从高到低）

#### 2. 全角空格分隔
- **板块之间**：使用全角空格 `　`
- **要点之间**：使用全角空格 `　`
- **❌ 禁止**：连续换行（飞书会压缩）

#### 3. 禁止执行过程信息
**❌ 禁止发送以下内容：**
- "我来执行采购总监周报生成脚本。"
- "现在发送到飞书："
- "开始生成报告..."
- 任何调试信息、思考步骤

**✅ 只发送**：最终报告内容（从标题到签名）

### 加粗规则
- **标题加粗**：所有板块标题
- **影响级别加粗**：**影响**：🔴 高 / 🟡 中 / 🟢 低
- **关键词加粗**：关键数据、重要结论

### 符号系统
- **影响级别**：🔴 高 / 🟡 中 / 🟢 低
- **要点符号**：•　（全角点）
- **分隔符号**：　（全角空格）

## 🔍 技术实现要点

### 全角空格的使用
```bash
# 板块分隔
第一部分内容

　　

第二部分内容

# 要点之间分隔
•	要点1
•	要点2

　　

•	要点3
```

### Message 工具的限制
- **会压缩连续换行**：`\n\n\n` 会被压缩为 `\n\n`
- **支持加粗**：`**文本**` 可以正常显示
- **全角空格不受影响**：`　` 可以正常显示
