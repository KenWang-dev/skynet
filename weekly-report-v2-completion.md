### 3. ✅ ROI计算逻辑（核心优化）
**级联计算方法：**
```
上游材料涨跌 X% × 占比 Y% = 中游产品成本变化 Z%
中游产品成本变化 Z% × 占比 W% = 下游产品成本变化 V%

示例：
CCL涨价10% × CCL占PCB材料50% = PCB成本上涨5%
PCB成本上涨5% × PCB占电子产品40% = 电子产品成本上涨2%

结论：供应商要求涨价5%不合理，实际应为2%，双方共担
```

**用途：**
- 快速评估成本涨幅的合理性
- 避免供应商随意转嫁成本
- 数据驱动的价格谈判

**状态**：✅ 已实现（cascade_roi_calc函数）

## 📊 核心优化点

### 1. 级联ROI计算

**功能：**
- 输入：上游涨跌%、上游占比、中游占比、采购量
- 输出：下游成本变化% + 金额影响

**示例输出：**
```
2.00% (金额影响: ¥20000)
```

**应用场景：**
- 价格谈判
- 成本分析
- 风险评估

---

### 2. 数据完整性检查

**功能：**
- 检查16个监控任务是否有数据
- 统计缺失项数量
- 计算完整性百分比

**示例输出：**
```
数据完整性：93.8% (缺失1项)
```

---

### 3. 价格传导分析

**功能：**
- 从原材料报告中提取价格数据
- 自动计算级联影响
- 生成谈判建议

**示例输出：**
```
### 价格传导分析（示例）

#### 场景1：CCL涨价对电子产品成本的影响

- **CCL涨价**：10%
- **CCL占PCB材料**：50%
- **PCB占电子产品材料**：40%
- **电子产品成本影响**：2.00% (金额影响: ¥20000)

**结论：** 如果供应商要求涨价5%，经分析实际应为2%，双方共担。
```
